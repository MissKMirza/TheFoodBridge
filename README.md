# 🍽️ The Food Bridge  
A platform connecting food donors with NGOs to reduce food waste and improve food accessibility.

---

## 🚀 Overview
Food waste and hunger coexist — The Food Bridge aims to bridge that gap by creating a system where individuals, restaurants, and organisations can donate surplus food, and NGOs can request and distribute it efficiently.

---

## 🎯 Problem Statement
- Large amounts of food are wasted daily  
- NGOs struggle to find timely food resources  
- Lack of a centralised system for coordination  

---

## 💡 Solution
The Food Bridge provides a **real-time platform** where:
- Donors can list surplus food  
- NGOs can browse and request available food  
- Coordination happens efficiently through a structured system  

---

## 🛠️ Tech Stack
**Frontend:** HTML, CSS, JavaScript  
**Backend:** Node.js, Express.js  
**Database & Auth:** Firebase  

---

## ⚙️ Features
- 🥗 Food donation listing system  
- 🔍 NGO browsing and request functionality  
- 🔐 User authentication (Firebase)  
- 🔄 Real-time database updates  
- 📱 Responsive UI for accessibility  

---

## 🧠 System Architecture
- RESTful APIs built using Express.js  
- Firebase is used for real-time data handling and authentication  
- Modular backend structure for scalability  

---

## 📊 Future Improvements

- 📦 Logistics and delivery tracking  
- 📈 Analytics dashboard for impact tracking
